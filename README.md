# Ctrl-Difuso-Onda-Verde-SEMAFOROS-con-Python-y-ESP32
Este proyecto es un control de sincronizacion de semaforos por control difuso, empleando python, ESP32, Contador de vehiculos y determinacion del estado del trafico con camaras IP.



Este repositorio lo voy a ir actualizando a medida que vaya avanzando con las distintas partes que conforman este proyecto.
